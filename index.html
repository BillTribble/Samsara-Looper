⚠️ Note for macOS users

Because Samsara is an experimental developer app, it is currently "unsigned." This means your Mac's built-in security will block the app from opening.

Use this quick Terminal trick to get things working:

  1. Open the Terminal app on your Mac (you can find it using Spotlight search).
  2. Type the following command, making sure to leave a single space at the end:
    xattr -d com.apple.quarantine
  3. Drag and drop the downloaded Samsara app or .vst3 file directly into the Terminal window. This will automatically fill in the correct file path for you.
  4. Press Enter.

You can now open the app or plugin!
