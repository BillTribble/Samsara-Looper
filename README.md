# 🔄 Samsara Looper

**A macOS-native retrospective looping workstation for musicians.**

Samsara is a real-time audio looper built as a native macOS app with an embedded web UI. It's designed for live performance and studio use, with features including:

- 🎵 Multi-track retrospective looping
- 🎛️ Real-time audio processing & FX
- 🔗 Ableton Link sync
- 🎹 MIDI control
- 🌐 Web-based UI via embedded browser
- 🎚️ Built-in mixer & session management
- 💾 Riff capture & history

## 📥 Downloads

Check the [Releases](https://github.com/BillTribble/Samsara-Looper/releases) page for the latest builds:

- 🖥️ **Standalone App** (.app) — Run Samsara as a standalone macOS application
- 🎛️ **VST3 Plugin** (.vst3) — Use Samsara inside your DAW

> Requires macOS 13+. Apple Silicon and Intel Macs supported.
